Why this project?

This landing page project is ideal for beginners — it uses only HTML and CSS with a small JavaScript enhancement for interactivity. You'll practice:

  • Creating responsive grid layouts
  • Using consistent spacing and typography
  • Designing simple UI components like cards, buttons, and forms
  • Preventing overlap using padding, gap, and media queries

Starter Checklist

  1. Plan your sections: header, hero, features, projects, footer
  2. Pick a color palette and limit to 2–3 primary colors
  3. Use CSS variables for consistent spacing
  4. Test in small screens and reduce column count
